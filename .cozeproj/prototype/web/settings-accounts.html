Chrome 调试配置说明

1

关闭所有 Chrome 窗口

确保 Chrome 完全退出，包括后台进程

2

使用调试模式启动 Chrome

在终端中执行以下命令（根据操作系统选择）：

macOS
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --remote-debugging-port=9222
Windows
"C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222
3

在 Chrome 中登录目标平台

依次打开小红书、抖音、TikTok、Instagram 等平台并登录账号

4

点击上方「连接浏览器」按钮

系统将自动检测 Chrome 调试端口并建立 CDP 连接

连接后，AI 将通过 CDP 协议操控你的浏览器会话完成平台操作。你的登录状态和会话 Cookie 会被复用，无需输入密码。所有操作均在你本地浏览器中执行，数据不会上传至服务器。

连接你的 Chrome 浏览器

1

关闭所有 Chrome 窗口

确保 Chrome 完全退出，包括系统托盘中的后台进程

2

复制启动命令并执行

在终端中粘贴以下命令启动 Chrome：

/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --remote-debugging-port=9222
3

登录目标平台

在打开的 Chrome 中登录你需要操作的平台账号

4

点击下方「检测连接」按钮

确认断开浏览器

断开浏览器连接后，以下功能将不可用：

  • 平台会话状态检测
  • CDP 自动化操作（发布内容、发送消息等）
  • 获客数据抓取
  • 浏览器截图

确认断开平台

断开该平台后，相关自动化操作将暂停。你可以随时重新检测连接。
